# Praticando Python: condicionais if, elif e else

## Sobre o curso
Neste módulo, pratiquei conceitos fundamentais de Python voltados para estruturas condicionais e lógica de programação.

## Conteúdos estudados
- Estruturas condicionais: `if`, `elif` e `else`
- Operadores lógicos: `and` e `or`
- Uso de variáveis para armazenar diferentes tipos de dados
- Tomada de decisão no código com base em condições
- Aplicação da lógica de programação em situações práticas

## O que aprendi
Ao longo deste curso, aprendi a utilizar estruturas condicionais para criar programas capazes de tomar decisões de acordo com diferentes cenários.

Também reforcei a importância de trabalhar com variáveis e escolher corretamente os tipos de dados, algo essencial para escrever códigos mais claros, organizados e eficientes.

Além disso, consegui enxergar melhor como esses conceitos se conectam com problemas do dia a dia de um desenvolvedor.
