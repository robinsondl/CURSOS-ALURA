# Praticando Python: laços for e while

## Sobre o curso
Neste módulo, pratiquei conceitos fundamentais de Python voltados para estruturas de repetição, com foco no uso dos laços `for` e `while`.

## Conteúdos estudados
- Laços de repetição com `for`
- Laços de repetição com `while`
- Diferença entre `for` e `while`
- Repetição de tarefas com base em intervalos e condições
- Aplicação de laços em exercícios práticos de lógica de programação

## O que aprendi
Ao longo deste curso, aprendi a utilizar estruturas de repetição para automatizar tarefas e executar blocos de código várias vezes de forma controlada.

Também reforcei o entendimento sobre quando utilizar `for`, mais indicado para repetições com quantidade definida, e quando utilizar `while`, mais adequado para situações baseadas em condições.

Além disso, consegui praticar a construção de soluções com mais lógica, organização e clareza, aplicando laços em diferentes cenários de programação.
